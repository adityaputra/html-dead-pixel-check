LCD Dead Pixel Check


This is a free simple tool to detect any dead pixels on your LCD screens. It is designed to work on any screen size. It works best on any web browser that supports full screen mode. If you find any bug, drop me an email at me@adityaputra.com.

How to test:

  1. Activate full screen mode on your web browser (This is usually can be enabled by using F11 key).
  2. Click on any buttons below to change color.
  3. Scroll down until your screen only displays blank page with background color that you've been selected.
  4. Cycle trough all colors
